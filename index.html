A Quick Note on Loss Functions

We often define a loss function $\mathcal{L}(\theta)$ to optimize model parameters. For a set of natural numbers, which we defined as $\N$, we can express the sum as $S = \sum_{i=0}^{n} i$.

A more complex example is the Euler's identity, which is a beautiful equation:

$$ e^{i\pi} + 1 = 0 $$

This shows how our custom command `\N` works, as well as both inline and display math rendering.
